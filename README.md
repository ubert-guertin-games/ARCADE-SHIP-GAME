# ARCADE SHIP GAME

## How to run this project ?

```
git clone https://github.com/ubert-guertin-games/ARCADE-SHIP-GAME.git

cd ARCADE-SHIP-GAME

bun install

bun dev

# Go to http://localhost:5173/
```
